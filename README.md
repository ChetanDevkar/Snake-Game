#  Snake Game

A classic Snake Game built using **HTML, CSS, and JavaScript**  .  
This game includes score tracking, high score storage, timer functionality, pause/resume feature, and smooth grid-based movement.

---

## 🚀 Features

- 🎮 Start and Restart Game
- 🏆 High Score saved using LocalStorage
- ⏱ Live Timer 
- ⏸ Pause/Resume feature (Press **P**)
- ⬆⬇⬅➡ Arrow key controls
- 💥 Wall collision detection
- 🍎 Random food generation
- 📱 Responsive grid layout

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Grid Layout & Custom Variables)
- JavaScript (DOM Manipulation, Game Logic, LocalStorage)

## 🎯 How to Play

1. Click **Start Game**
2. Use Arrow Keys to control the snake:
   - ⬆ Arrow Up
   - ⬇ Arrow Down
   - ⬅ Arrow Left
   - ➡ Arrow Right
3. Press **P** to Pause/Resume
4. Avoid hitting the wall
5. Eat food to grow and increase score

---

## 💾 High Score System

The game uses **LocalStorage** to store high scores.
Even after refreshing the page, your high score remains saved.

---

## 🧠 Game Logic Overview

- The board is created dynamically using JavaScript grid blocks.
- Snake movement is controlled using keyboard events.
- Food is generated at random positions.
- Score increases by 10 points per food.
- Game ends if the snake hits the wall.
- Timer runs every second.
